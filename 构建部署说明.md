# 数可信终端 — 构建与部署文档

本项目已迁移到 **Wails v2 + Go** 架构。前端 Vue 3 + Vite 编译进 `frontend-assets/`，由 `main.go` 通过 `//go:embed` 嵌入 Go 二进制；后端 Gin 监听 `:3001`，前端通过 HTTP 调用。

## 一、环境准备

### 通用依赖

- **Go**: 1.26.x 或更高
- **Node.js**: 18.x 或更高
- **Yarn**: 1.22.x（项目使用 yarn，不要用 npm）。Node 16+ 直接 `corepack enable` 即可
- **Wails CLI**: `go install github.com/wailsapp/wails/v2/cmd/wails@latest`

### 平台 SDK

- **macOS**: `xcode-select --install`
- **Windows**: 安装 [WebView2 Runtime](https://developer.microsoft.com/zh-cn/microsoft-edge/webview2/)
- **Linux (Ubuntu/Debian)**:
  ```bash
  sudo apt-get install -y libgtk-3-dev libwebkit2gtk-4.0-dev build-essential pkg-config
  ```

### 验证

```bash
go version          # go1.26.x
node --version      # v18+
yarn --version      # 1.22.x
wails doctor        # 检查 wails 依赖
```

## 二、开发模式

```bash
wails dev
```

会自动：
1. 跑 `yarn install --production=false` 装前端依赖
2. 跑 `yarn build` 出一份初始 `frontend-assets/`
3. 起 Go 后端（Gin 监听 `:3001`）+ Vite 开发服务器（默认 `:5173`）
4. 在桌面 webview 里加载 Vite，HMR 实时生效

什么场景需要 Ctrl+C 重启 `wails dev`：改了 `wails.json` / `vite.config.ts` / `package.json` 依赖。

## 三、生产打包

```bash
wails build
```

输出位置：

- macOS: `build/bin/data-asset-scan.app`
- Windows: `build/bin/data-asset-scan.exe`
- Linux: `build/bin/data-asset-scan`

构建步骤（wails 自动按顺序执行）：
1. `yarn install --production=false`
2. `yarn build`（vite 把前端打到 `frontend-assets/`）
3. `go build`（embed `frontend-assets/`，生成单文件二进制）

## 四、独立运行后端（不开窗口）

调试时可以只起 HTTP 后端：

```bash
go run .
```

此时浏览器侧不可用（webview 没起），仅供 `curl` / Postman 验证 API。

## 五、目录结构

```
.
├── main.go                   # Wails 入口：embed 前端 + 启动 Gin + Wails App
├── internal/
│   ├── httpd/                # HTTP 路由（router.go 注册全部）
│   ├── repository/           # SQLite 数据访问
│   ├── scanner/              # 扫描引擎
│   └── models/               # 领域模型
├── frontend_real/            # Vue 前端源码（Vite root）
│   ├── index.html            # Vite 入口模板
│   ├── main.ts               # Vue 应用入口
│   ├── views/                # 业务页面
│   └── services/             # 前端服务（含 api.ts）
├── frontend-assets/          # Vite 构建产物，被 main.go embed
├── wails.json
├── vite.config.ts            # root=frontend_real, outDir=frontend-assets
└── package.json
```

## 六、清空数据

数据库默认位于：

- macOS / Linux: `~/.local/share/data-asset-scan/data.db`
- Windows: `%APPDATA%/data-asset-scan/data.db`

删除该文件即重置（启动时会重新初始化 schema）。
